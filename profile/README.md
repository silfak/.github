# 🌿 Welcome to Kelompok 6 Workspace!

Kami adalah tim pengembang di balik **SILFAK** (Sistem Laporan Fasilitas Fakultas Ilmu Komputer). Organisasi ini merupakan pusat kolaborasi (*multi-repo environment*) untuk seluruh kode, aset, dan manajemen proyek kami.

## 🎯 Our Mission & Focus
Membawa tema **“Sustainable Campus & Green Lifestyle”** (Opsi 11), platform SILFAK dibangun sebagai wujud nyata dukungan kami terhadap *Sustainable Development Goals* (SDG), yang beririsan pada:
* 🏙️ **SDG 11 (Sustainable Cities and Communities):** Mewujudkan lingkungan kampus yang aman, inklusif, dan tangguh melalui manajemen fasilitas yang tertata.
* ♻️ **SDG 12 (Responsible Consumption):** Mencegah pemborosan energi (listrik, air) dengan pelaporan *real-time* dan *data-driven decision making*.

## 📂 Our Repositories
Proyek SILFAK dipecah menjadi beberapa *repository* utama untuk menjaga modularitas:

* 🖥️ **[`silfak-frontend`](https://github.com/silfak/web-frontend)**: *Client-side application* yang dibangun menggunakan React 19, Vite, Tailwind CSS v4, dan Shadcn UI.
* 🗄️ **[`silfak-backend`](https://github.com/silfak/api-backend)**: *REST API server* yang dibangun menggunakan Node.js (Express), Drizzle ORM, dan PostgreSQL.
* 📄 **[`docs`](#)**: *(Optional)* Berisi kumpulan dokumen pendukung seperti PRD (Product Requirements Document), API Schema, dan desain UI/UX.
